## instalacja

pip install django

albo

uv add install django


## tworzenie projektu

django-admin startproject nazwa_projektu

## uruchomienie serwera

python manage.py runserver


## tworzenie aplikacji:

 python manage.py startapp nazwa_aplikacji


## cwiczenia:

### 1 routing i views
